# Dashboard

A simple React dashboard application.

## Features

- Responsive layout
- Real-time data visualization
- User-friendly interface

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

The dashboard will open at `http://localhost:3000`.

## Build

```bash
npm run build
```

Creates an optimized production build in the `build` folder.

## Technologies

- React
- CSS3
- JavaScript ES6+
